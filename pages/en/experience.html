Work experience:

5 June - 30 August 2017 (3 months) Web / Business Intelligence internship - IT-CE

Location: Paris

IT-CE is the IT department of BPCE group. BPCE result from a merge of two banks (Banque populaire and Caisse d’épargne). IT-CE ideals with IT of the company from architecture to network including research & development.

My internship took place in the unit “Architecture and innovation” which is in charge of software architecture for their proprietary software as well as commercial software.

The first mission was to develop decision-making dashboards using PowerBi and MSSQL. The dashboards displayed in a global and almost real time, all creations or modifications of software in the company.

The second mission was to develop an automated website for documentation: Writen in C# (ASP.NET) the website collects Word document, analyses it and store its content in a database and a JSON file, allowing to see this content later in a web page. The website goal was to be easy to use and read (Chapters in the document were replaced by clickable drop-down menus, the interface was developed to be able to get straight to the point).

During this internship I had the opportunity to create also an API to dynamically extract Word content. This API is open source and available on my GitHub profile



1st - 30 August 2016

(1 month)

 Internship in an IT department - Centre Hospitalier Princesse Grâce

Location: Monaco

The hospital Princesse Grâce is the major public hospital in Monaco. It has a capacity of 759 bed shared in three buildings.

I did my internship in the IT department, which was in charge of the network, security and software for the hospital.

During my internship, I developed a software for the Nao robot. It was used in the geriatric department for old people with memory disabilities.

When the robot recognises one of the patients, it tries to interact with him/her, and then propose to do some memory games, or to dance.

The robot is used by medical staff, so the software is designed to work without a computer: to update recognised patients, the staff only needs to touch some sensors during power up process, and indicates whose patient to add or remove from the robot’s memory.
